F.A.Q.

  • What is Progressive Web Application HQ?

    Progressive Web Application HQ is a web site to explain the main characteristics of progressive web apps at the same time it serves as a hub for gathering documentation and resources. Progressive Application HQ is maintained by Mozilla WADI initiative.

  • What is a Progressive Web Application?

    It is a regular website using certain technologies that enable it to behave like a native application under certain platforms such as Android or Firefox OS.
