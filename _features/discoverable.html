As soon as you start writing progressive web applications you want the World to know about them. If a user visits your application in a modern platform, all your web site's progressive super-powers simply emerege but, what if the user never heard about you? Progressive Web Applications are discoverable because they allow search engines and Web indices to find your application and highlight their capabilities.
