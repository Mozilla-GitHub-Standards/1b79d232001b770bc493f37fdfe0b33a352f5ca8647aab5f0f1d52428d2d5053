Technologies

Progressive Web Applications use the correct set of technologies to leverage UX in the Web…

  • Service Workers

    Cache network resources and enable super fast access to them.

  • Web Application Manifest

    Distribute your applications through the ultimate platform: the Web.

  • Push API

    Stand your application out and draw the user attention.

  • Add to homescreen

    Allow the web application to become a first-class citizen in your device.

  • Background Sync

    Perform tasks on regaining connection.

  • App Shell

    Use app shell architecture to reach almost instant first draw.

  • Media Query

    Adapt your design to bring the best user experience.
